# CONTROLS

## Movement
- **A / Left Arrow**: move left
- **D / Right Arrow**: move right
- **SPACE**: jump (supports coyote + jump buffer + jump cut)

## Combat
- **J**: light combo (3-hit chain)
- **K**: heavy attack

## Skills
- **Q**: Ascending Rupture
- **W**: Shadow Step
- **E**: Flow Blade
- **R**: Freezing Prism
- **F**: Overload Core

## UI / Debug
- **ESC**: pause/settings
- **F3**: debug overlay
- **F5**: debug +XP/+Gold
- **F6**: debug spawn mini-boss
- **F7**: debug spawn enemy wave
- **F8**: debug manual save

## Rebinding
- Open **ESC > Rebind Controls**, click action, then press desired key.
