# Controls

## Keyboard

- **Move**: `W/A/S/D` or `Arrow Keys`
- **Jump**: `Space` or `Z` or `Up Arrow`
- **Attack**: `X` or `K` (Light Combo: Mash, Heavy: Hold)
- **Dash**: `C` or `L` or `Shift`
- **Skills**: `1, 2, 3, 4, 5`
- **Interact**: `E`
- **Pause**: `ESC`

## Gamepad (Generic)

- **Move**: `D-Pad` or `Left Stick`
- **Jump**: `A` / `X` (South Button)
- **Attack**: `X` / `Y` (West Button)
- **Dash**: `B` / `O` (East Button)
- **Skills**: `Shoulder Buttons` / `Triggers`

## Skills Details

1. **Ascending Rupture (Key 1)**: Vertical launcher, high damage.
2. **Shadow Step (Key 2)**: Quick intangible dash, cleanses slows.
3. **Flow Blade (Key 3)**: Multi-hit sustained damage.
4. **Freezing Prism (Key 4)**: Crowd control (Freeze).
5. **Overload Core (Key 5)**: Massive AoE damage (Burst).
